Политика конфиденциальности

Дата вступления в силу: 02.07.2025

1. Какие данные мы собираем:

  • ФИО, номер телефона, адрес электронной почты;
  • Паспортные данные, ИНН, данные СТС, ПТС, ВУ и т.д.;
  • Информация о транспортном средстве (марка, модель, VIN и прочее);
  • Банковские реквизиты (для агентов: номер карты, ФИО владельца счёта, наименование банка);
  • Данные о полисах и действиях в приложении;
  • Файлы, такие как копии документов (паспорт, водительское удостоверение и пр.).

2. Как мы используем данные:

  • Для регистрации и авторизации пользователей;
  • Для оформления, хранения и отображения электронных полисов;
  • Для расчета страховых премий и отображения баланса агента;
  • Для передачи банковских реквизитов в страховую компанию с целью выплаты вознаграждения агенту вручную;
  • Для предоставления поддержки и уведомлений;
  • Для соблюдения требований законодательства (например, в сфере страхования).

3. Хранение и защита данных:

  • Данные пользователя сохраняются на устройстве с помощью локальной базы данных;
  • Аутентификационные токены сохраняются на устройстве и используются только для проверки входа в систему;
  • Для передачи данных используется защищённое соединение HTTPS (SSL);
  • Банковские реквизиты не сохраняются на устройстве и передаются только на сервер по защищённому соединению;
  • Мы соблюдаем требования законодательства о защите персональных данных.

4. Третьи лица:

Мы не передаём ваши персональные данные третьим лицам без вашего согласия, за исключением случаев, предусмотренных законом (например, передача в страховые компании или госорганы при необходимости).

5. Удаление данных:

Вы можете запросить удаление своих данных, отправив обращение в службу поддержки.

6. Платёжные данные

Оплата полисов ОСАГО в приложении ALMA осуществляется через внешний защищённый платёжный шлюз по deeplink-ссылке. Приложение не собирает, не обрабатывает и не хранит данные платёжных карт пользователей.

ALMA получает только информацию о статусе оплаты (успешно/неуспешно) от платёжной системы. Это необходимо для завершения оформления полиса. Все платёжные операции проходят в соответствии с политикой безопасности платёжного сервиса.

7. Банковские реквизиты агентов

Для выплаты агентского вознаграждения страховая компания может запросить у агента номер карты, имя владельца счёта и наименование банка. Эти данные запрашиваются в приложении ALMA и передаются напрямую страховой компании по защищённому каналу. Приложение не хранит и не использует банковские реквизиты для автоматических переводов и не осуществляет финансовые транзакции.

8. Контакты:

Если у вас возникли вопросы по поводу конфиденциальности, напишите нам: support@alma.insurance
